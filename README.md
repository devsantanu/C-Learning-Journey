# C Learning Journey

A comprehensive record of my C programming learning journey, covering fundamental concepts, practice exercises, and mini-projects as I build a strong foundation for my BTech CSE (Cloud Computing & AI) degree.

## About

This repository documents my progress in learning the C programming language — from basic syntax and logic building to small interactive programs. Each folder represents a topic or a project I've worked on.

## Repository Structure

| Folder | Description |

| `Games/` | Interactive console games (Number Guessing Game, Snake Water Gun) — see its own [README](./Games) for details |
| `Grade system/` | Program to calculate and display grades based on marks |
| `Profit Loss/` | Program to calculate profit or loss based on cost and selling price |

*(More folders will be added as I continue learning new topics.)*

## How to Run Any Program

Most programs can be compiled and run using GCC:

```bash
gcc filename.c -o filename
./filename
```
