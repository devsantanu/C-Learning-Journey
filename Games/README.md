# C Logic Games Collection

This folder contains various logic-based and interactive console games developed in C language. The primary goal of these mini-projects is to strengthen core programming concepts, including control structures, random number generation, loops, and robust user input validation.

## Games Index

1. **[Number Guessing Game](./Number_Guessing_Game)** — Guess a randomly generated number between 1 and 100 with hints and an attempt counter.
2. **[Snake, Water and Gun Game](./Snake_Water_Gun_Game)** — A Rock-Paper-Scissors variant where the player competes against the computer.

## Common Concepts Practiced

- Random number generation (`rand()`, `srand()`)
- Input validation and error handling
- Loops (`do-while`) for repeated gameplay
- Conditional logic for decision-making
